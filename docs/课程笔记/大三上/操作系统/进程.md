## 进程的概念与特征

OS如何处理程序/作业的发展

- 一次装入并运行一个“作业”
- 铜焊丝装入多个“程序”，并发执行
- 同时装入多个程序，并发执行，仅当等待I/O时切换程序执行
- 同时装入多个程序，并发执行，程序切换条件：执行时间长度到**或者**I/O等待

进程概念的提出（Multics,1964）

不同角度下描述的进程

1. 一个正在执行的程序
2. 计算机中正在运行的程序的一个实例
3. 可以分配给处理器，并由处理器执行的一个实体
4. 由一个顺序执行的代码段、一个当前状态和一组相关系统资源所刻画的活动单元
