## 有符号数

- 真值：带符号的数

- 机器数：符号数字化的数

```
+0.1011或0.1001
-0.1011
+1100或1100
-1100
```

> 非特殊说明，默认二进制表示<br>
> 位数不是的倍数机器数一般比真值多一位符号位
